变量

变量的命名规则

变量的名字又叫做“标识符”，必须以字母、下划线（ _）或者美元符号（ $）开头，并且区分大小写。

JavaScript三种变量声明方式：

  • var：声明一个变量
  • let：声明一个块作用域的局部变量，可赋一个初始值
  • const：声明一个块作用域的只读常量

声明变量

// 若没有为变量赋初始值，则值默认为 undefined；
let a;
console.log(a); // undefined


// 若没有声明变量而直接使用，会抛出 ReferenceError误；
console.log(b); // ReferenceError: b is notdefined


// 当变量值为 undefined时，布尔值环境会当做 false数值环境会当做 NaN；
var a;
if (!a) {
    console.log('a为undefined'); // a为undefined
}
console.log(a + 1); //NaN


// 当变量值为 null时，布尔值环境会当做 false，数值境会当做 0；
let a = null
if (!a) {
    console.log('a为null'); // a为null
}
console.log(a + 1); // 1

声明提升

//变量提升
var a = 1;
等价于
var a = undefined
a = 1


/*
 函数提升
*/
//函数声明
f(); // 'hi leo'
function f(){
    console.log('hi leo');
};

// 函数表达式
g();   // Uncaught TypeError: g is not a function
var g = function(){   // 换成 let 声明也一样
    console.log('hi leo');
}

变量作用域

全局变量（var）：即声明在函数之外，当前文档所有地方都可以访问；
局部变量（let）：即声明在函数内部，仅在当前函数内可以访问；
常量（const）：在声明时必须赋值，之后相同作用域中不能赋值也不能重新声明；

数据类型

JavaScript中一共分为7中不同类型：

  • 六种基本数据类型：
    • 1.Boolean : 布尔值，true和false;
    • 2.null : 对大小写敏感（即 null/ Null/ NULL完全不同）;
    • 3.undefined : 空类型，变量未定义时候的值；
    • 4.Number : 数值类型，如 100;
    • 5.String : 字符串类型，如’hi pingan‘;
    • 6.Symbol（ES6新增） : 表示一种唯一且不可变的数据;
  • 以及Object对象类型
