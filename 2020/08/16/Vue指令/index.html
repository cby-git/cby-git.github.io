Vue指令

指令本质就是自定义属性，Vue中的指令都是以 v- 开头。

内容绑定，事件绑定

v-text：设置标签内容

  • 默认写法会替换全部内容，使用差值表达式 {{ }} 可以替换指定内容。

  • 内部支出写表达式

  • 如果数据中有HTML标签会将html标签一并输出

  • 注意：此处为单向绑定，数据对象上的值改变，插值会发生变化；但是当插值发生变化并不会影响数据对象的值

<body>
    <div id="app">
        <h2 v-text='message'></h2>
        <h2>{{info}} 上海</h2>
    </div>
    <!-- 导入vue 开发环境版本，包含了有帮助的命令行警告 -->
    <script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
    <script>
        var app = new Vue({
            el: '#app',
            data: {
                message: 'Vue 文本指令1',
                info:'Vue 文本指令2'
            }
        })
    </script>
</body>

v-html：设置标签的innerHTML

  • 用法和v-text 相似 但是他可以将HTML片段填充到标签中

  • 可能有安全问题, 一般只在可信任内容上使用 v-html，永不用在用户提交的内容上

  • 它与v-text区别在于v-text输出的是纯文本，浏览器不会对其再进行html解析，但v-html会将其当html标签解析后输出。

    <body>
  <div id="app">
      <h2 v-html='message'></h2>
      <p v-html='info'></p>
  </div>
  <!-- 导入vue 开发环境版本，包含了有帮助的命令行警告 -->
  <script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
  <script>
        var app = new Vue({
            el: '#app',
          data: {
                message: 'Vue 文本指令1',
                info:'<h2>上海</h2>'
          }
      })
  </script>
</body>

    v-on基础

  • 用来绑定事件的，时间名不需要写on

  • 指令可以简写为@。如：v-on:click 缩写为 @click;

  • 绑定的方法定义在methods中

  • 方法内部通过this关键字可以访问定义在data中的数据

显示切换，属性绑定

v-show：根据真假切换元素的状态

  • 原理是修改元素的display，实现显示隐藏
  • 指令后面的内容最终都会解析为布尔值
  • 值为true元素显示，值为false元素隐藏
  • 数据改变后对应的显示状态会同步更新
<body>
    <div id="app">
        <input type="button" value="切换显示状态" @click='changeIsShow'>
        <input type="button" value="累加年龄" @click='addAge'>
        <img v-show='isShow' src="image/apple.png" alt="" srcset="">
        <img v-show='age>=18' src="image/orange.png" alt="" srcset="">
    </div>
    <script type="text/javascript" src="js/vue.js"></script>
    <script type="text/javascript">
        var vm = new Vue({
            el: '#app',
            data: {
                isShow:false,
                age:17
            },
            methods: {
                changeIsShow:function(){
                    this.isShow = !this.isShow;
                },
                addAge:function(){
                    this.age++
                }
            }
        });
    </script>
</body>

v-if：根据表达式的真假切换元素的显示状态

  • 本质通过操作dom元素切换显示状态
  • 表达式为true，元素存在于dom树中，为flase，从dom树中删除
  • 频繁切换v-show，反之使用v-if,前者的切换消耗小
<body>
    <div id="app">
        <input type="button" value="切换状态" @click='toggIsShow'>
        <input type="button" value="切换状态2" @click='addAge'>
        <p v-if='isShow'>if指令</p>
        <h2 v-if='temperatrue>=30'>if指令</h2>
    </div>
    <script type="text/javascript" src="js/vue.js"></script>
    <script type="text/javascript">
        var vm = new Vue({
            el: '#app',
            data: {
                isShow:false,
                temperatrue:20
            },
            methods: {
                toggIsShow:function(){
                    this.isShow = !this.isShow;
                },
                addAge:function(){
                    this.temperatrue++
                }
            }
        });
    </script>
</body>

v-bind：为元素绑定属性

  • 完整写法是v-bind:属性名
  • 简写的话可以直接省略v-bind，只保留属性名
  • 需要动态的增删class建议使用对象的方式
<body>
    <div id="app">
        <div class="div" :class="isActive ? 'active' : ''" @click='toggActive'>
            1234
        </div>
    </div>
    <script type="text/javascript" src="js/vue.js"></script>
    <script type="text/javascript">
        var vm = new Vue({
            el: '#app',
            data: {
                isActive:false
            },
            methods: {
                toggActive:function(){
                    this.isActive = !this.isActive;
                }
            }
        });
    </script>
</body>

列表循环，表单元素绑定

v-for：根据数据生成列表结构

  • 经常数组和v-for一起使用
  • 语法(item,index) in 数据
  • item和index可以结合其他指令一起使用
  • 数组长度的更新会同步到页面上，是响应式的
  • 不推荐同时使用 v-if 和 v-for
  • 当 v-if 与 v-for 一起使用时，v-for 具有比 v-if 更高的优先级。
<ul id="example-1">
   <!-- 循环结构-遍历数组  
    item 是我们自己定义的一个名字  代表数组里面的每一项  
    items对应的是 data中的数组-->
  <li v-for="item in items">
    {{ item.message }}
  </li> 

</ul>
<script>
 new Vue({
  el: '#example-1',
  data: {
    items: [
      { message: 'Foo' },
      { message: 'Bar' }
    ]，

  }
})
</script>

v-on补充

  • 事件绑定的方法写成函数的形式时，可以传入自定义参数

  • 定义方法时需要定义形参来接收传入的实参

  • 事件的后面跟上 .修饰符 可以对事件进行限制

  • .enter 可以限制触发的按键为回车

  • 事件的修饰符有多种

    <body>
  <div id="app">
      <div>{{num}}</div>
      <div>
          <!-- 如果事件直接绑定函数名称，那么默认会传递事件对象作为事件函数的第一个参数 -->
          <button v-on:click='handle1'>点击1</button>
          <!-- 2、如果事件绑定函数调用，那么事件对象必须作为最后一个参数显示传递，
                 并且事件对象的名称必须是$event 
            -->
          <button v-on:click='handle2(123, 456, $event)'>点击2</button>
      </div>
  </div>
  <script type="text/javascript" src="js/vue.js"></script>
  <script type="text/javascript">
        var vm = new Vue({
            el: '#app',
          data: {
              num: 0
          },
          methods: {
                handle1: function(event) {
                    console.log(event.target.innerHTML)
              },
                handle2: function(p, p1, event) {
                    console.log(p, p1)
                    console.log(event.target.innerHTML)
                    this.num++;
              }
          }
      });
  </script>
</body>

    Vue 不推荐我们操作DOM 为了解决这个问题，Vue.js 为 v-on 提供了事件修饰符
    修饰符是由点开头的指令后缀来表示的使用修饰符时，顺序很重要；相应的代码会以同样的顺序产生。因此，用 v-on:click.prevent.self 会阻止所有的点击，而 v-on:click.self.prevent 只会阻止对元素自身的点击。

    <!–hexoPostRenderEscape:

    <!-- 阻止单击事件继续传播 -->
    <a v-on:click.stop="doThis"></a>

<!– 提交事件不再重载页面 –>
<form v-on:submit.prevent="onSubmit"></form>

<!– 修饰符可以串联 即阻止冒泡也阻止默认事件 –>
<a v-on:click.stop.prevent="doThat"></a>

<!– 只当在 event.target 是当前元素自身时触发处理函数 –>
<!– 即事件不是从内部元素触发的 –>
<div v-on:click.self="doThat">…</div>

:hexoPostRenderEscape–>

v-model:便捷的设置和获取表单元素的值

  • 绑定的数据会和表单元素值相关联

  • 绑定数据←→表单元素的值

  • 限制在 <input>、<select>、<textarea>中使用

    <body>
  <div id="app">
      <input type="button" value="修改message" @click='setM'>
      <input type="text" v-model="message" @keyup.enter='getM'>
      <h2>{{ message }}</h2>
  </div>
  <script src="js/vue.js"></script>
  <script>
        var mask = new Vue({
            el:'#app',
          data:{
                message:'哈哈哈'
          },
          methods:{
                getM:function(){
                    alert(this.message)
              },
                setM:function(){
                    this.message = '呵呵呵'
              }
          }
      })
  </script>
</body>
